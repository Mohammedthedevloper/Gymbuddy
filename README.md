GymBuddy – Fitness Tracking App

Overview  
GymBuddy is an Android fitness app designed to connect clients, trainers, and admins through a single, easy-to-use platform.  
The app helps users track workouts, monitor progress, and stay consistent while allowing admins to verify and manage accounts securely.  

------------------------------------------------------------

Features  
- Firebase Authentication – secure login and registration  
- Admin Verification – only admins can approve new users  
- Workout & Progress Tracking – record sets, reps, and weights  
- Weekly Summary Dashboard – view body weight and performance data  
- Real-time Cloud Sync – data instantly updates for all users  
- Trainer Dashboard – manage and monitor client progress  

------------------------------------------------------------

Future Enhancements  
- Calendar integration for workout scheduling  
- Discovery Health integration for reward tracking  
- Menstrual and health tracking  
- Auto-run detection and sound-level monitoring for hearing protection
